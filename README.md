# Pulsar Detection

## How to use

#### In order to clone and run this desktop application you'll need Git and Node.js installed on your computer. If you do not have them, download them here:
- [Git](https://git-scm.com/downloads)
- [Node.js](https://nodejs.org/en/download)

#### After that from your command line:
```console
# Clone the repository
git clone https://github.com/HMate02/Pulsar_Detection.git
# Go into the directory
cd Pulsar_Detection
# Install dependencies
npm install
# Run the desktop application
npm start
```
